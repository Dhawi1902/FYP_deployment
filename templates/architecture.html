Model Architecture

VAE Architecture

Variational Autoencoders (VAEs) are a type of generative model that uses neural networks to learn a compressed representation of input data.

EDGAN Architecture

Encoder-Decoder GAN (EDGAN) is a type of Generative Adversarial Network that includes an encoder and a decoder for generating images.

RESNET50 Architecture

ResNet50 is a residual neural network that is 50 layers deep. It is used for image recognition and classification tasks.
